L1 稀缺资源品区

TV ≥ 95--

层级定义

全球限量供应、稀有原料、不可复制的地域性珍品。退役军人专属采购渠道，平台严格筛选准入。

TV计算：质量评分×40% + 稀缺指数×25% + 供需比×15% + 评价×10% + 溯源×10%

L2 溢价品区

85 ≤ TV < 95--

层级定义

品牌溢价明显、文化附加值高、市场认可度强的优质商品。价值增长空间明确，适合品质消费与礼品场景。

强调品牌力与市场口碑，溢价来源于消费者信任与品牌沉淀。

L3 增值品区

75 ≤ TV < 85--

层级定义

具有持有增值潜力的商品，如年份茶、陈酿酒、限量工艺品。链上溯源确权保障投资价值，时间复利驱动增值。

增值因子：时间沉淀×稀缺增长×市场认可度提升。区块链存证确保权属清晰。

🔗 产品入口

L4 自报价值品区

60 ≤ TV < 75--

层级定义

商家自主定价、平台公开核验。强调定价权回归生产者，消费者可查看完整的成本结构与溯源链条，实现真正的价格透明。

核心机制：商家自报成本+利润 → 平台AI核验合理性 → 区块链公示 → 消费者知情选择

🔗 产品入口

L5 保值品区

45 ≤ TV < 60--

层级定义

资产保值型商品，价格稳定、需求刚性。退役军人信用背书，适合稳健消费与长期复购。抗通胀属性突出。

保值特性：需求刚性×退役军人信用加成×供应链稳定度。价格波动幅度控制在±5%以内。

🔗 产品入口

L6 优质农产品帮扶区

30 ≤ TV < 45--

层级定义

乡村振兴消费帮扶专区。产地直发、品质保障，以消费促帮扶。退役军人管理的生态农庄为核心供应源。

帮扶机制：消费即帮扶 → 交易额3%注入帮扶基金 → JC功勋币奖励 → 退役军人优先品控
